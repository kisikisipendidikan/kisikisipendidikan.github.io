Materi: Administrasi Sistem Jaringan - FTP

Jawaban:

Server yang berfungsi sebagai pemberi akses/pertukaran transfer data antara dua komputer atau lebih adalah B. FTP Server.

Penjelasan:

FTP Server memiliki kepanjangan File Transfer Protocol. Yang berarti protokol untuk transfer berkas (file). Itu bermakna bahwa FTP Server adalah jenis server yang berfungsi untuk melakukan tindakan transfer pertukaran data (exchange) dengan akses antara dua perangkat komputer atau lebih.

Jadi, opsi yang benar adalah B. FTP Server.

19. Bagaimana cara kerja web server dan apa saja yang bisa dilakukan oleh E - mail server


Jawaban:

Sederhananya tugas web server adalah untuk menerima permintaan dari klien dan mengirimkan kembali berkas yang diminta oleh klien tersebut. Perangkat lunak web server terdapat pada komputer server, dan di komputer ini pula data-data website tersimpan dengan rapih.

Yang bisa dilakukan oleh Email Server sebenarnya banyak dan tentu saja jika developer ingin membuat suatu fitur maka mereka tinggal menambahkan beberapa kode yang nantinya fungsinya sesuai yang mereka harapkan. Beberapa yang dapat dilakukan sebuah server email adalah mengirim & menerima email, meneruskan email dari 1 pengguna ke pengguna lainnya, dan juga yang sekarang sudah umum dan masih terus ditingkatkan adalah Spam Protection dimana server otomatis memisahkan email yang dinilai sebagai email spam menurut respon kebanyakan pengguna.

Mungkin ke depannya akan ditambahkan fungsi untuk keamanan dari virus, spyware, malware, dsb. yang akan sangat membantu pengguna supaya tetap aman.

33. Server Adalah web server adalah Server FTP adalah Server DHCP Adalah


Mapel : TI
Materi : Server

--Soal--

Server Adalah
Web server adalah
Server FTP adalah
Server DHCP Adalah

--Jawaban--

Server adalah suatu sistem komputer yang menawarkan jenis layanan tertentu dalam suatu jaringan.

Web server adalah suatu perangkat lunak (software) yang memberikan perintah berbasis data yang berfungsi menerima permintaan dari http atau https pada klien web terkenal.

Server FTP adalah protokol internet yang berjalan dalam suatu lapisan aplikasi yang berfungsi untuk tukar menukar berbagai data antara klien dan server dalam 1 jaringan.

Server DHCP adalah adalah suatu protocol pada jaringan komputer yang dapat memberikan atau meminjamkan atau memberi akses IP address terhadap host yang berada dalam suatu jaringan secara otomatis.

37. untuk dapat mengirimkan email dari mail server, maka mail server harus memiliki? ​


Jawaban:

# client-server

Penjelasan:

Ada tiga protocol utama dalam mengirimkan email melalui mail server, yaitu protocol SMTP (Simple Mail Transfer Protocol), POP3 (Post Office Protocol V3), dan IMAP SMTP. SMTP lebih banyak digunakan untuk menampung data serta mengirimkan email ke alamat email lain yang ingin dituju. Protocol POP3 dan protocol IMAP lebih banyak digunakan untuk memudahkan user dalam membaca dan mengakses email secara remote.
