Jawaban:

Berikut jawabannya dalam bahasa C++:

#include<stdio.h>

void printAllData(int listData[3][5],int row,int column)

{

printf("Print Data\n");

for(int i=0; i<row; i++)

{

 for(int j=0; j<column; j++)

 {

  printf("%-8d ",listData[i][j]);

 }

 printf("\n");

}

}

void multiplyAllDataByN(int listData[3][5],int row,int column, int n)

{

for(int i=0; i<row; i++)

{

 for(int j=0; j<column; j++)

 {

  listData[i][j] *=n;

 }

}  

}

int main()

{

int row = 3, column = 5;

int listData[3][5];

for(int i=0; i<row; i++)

{

 for(int j=0; j<column; j++)

 {

  scanf("%d",&listData[i][j]);

 }

}

printAllData(listData,row,column);

int n = 3;

multiplyAllDataByN(listData,row,column,n);

printAllData(listData,row,column);

getchar();

return 0;

}

Penjelasan:

Program ini pertama-tama akan mendapatkan input user berupa array multidimensi 3x5 bertipe integer. Kemudian method printAllData mencetak seluruh isi array untuk menunjukkan nilai sebelum diubah lalu seluruh isi data array dikalikan sebesar n dimana n = 3 seluruhnya. 30. jelaskan maksud kode berikut int array 1 [5] = {7,6,9,1,16} ;​


Jawaban:

Kode tersebut merupakan inisialisasi dan deklarasi dari sebuah array dengan nama "array1" yang memiliki ukuran 5 dan berisi nilai-nilai {7, 6, 9, 1, 16}.

Arti dari kode tersebut adalah sebagai berikut:

- "int" merupakan tipe data dari elemen-elemen dalam array, dalam hal ini bertipe integer (bilangan bulat).

- "array1" adalah nama dari array tersebut. Nama ini bisa digunakan untuk mengakses elemen-elemen dalam array.

- "[5]" menunjukkan bahwa array memiliki ukuran 5, artinya terdiri dari 5 elemen.

- "{7, 6, 9, 1, 16}" adalah nilai-nilai yang disimpan di dalam array. Dalam hal ini, array memiliki 5 elemen dengan nilai berturut-turut 7, 6, 9, 1, dan 16.

Jadi, dengan kode tersebut, kita telah membuat dan menginisialisasi array "array1" dengan 5 elemen yang berisi nilai-nilai tertentu. Misalnya, kita bisa mengakses nilai 6 dengan menggunakan "array1[1]", dan nilai 16 dengan "array1[4]".
