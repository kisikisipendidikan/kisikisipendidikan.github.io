1. TENTUKAN HASIL DARI : (20202019 x 20192020) - (20202020 x 20192019) =... SILAHKAN DISKUSIKAN, BERI TANGGAPAN DAN SERTAKAN PULA SUMBER LITERATURNYA..


Penjelasan dengan langkah-langkah:

Misal,

a = 20192019

a + 1 = 20192019 + 1

a + 1 = 20192020

b = 20202019

b + 1 = 20202019 + 1

b + 1 = 20202020

(20202019 x 20192020) - (20202020 x 20192019)

= (b x (a + 1)) - ((b + 1) x a)

= (ab + b) - (ab + a)

= ab + b - ab - a

= ab - ab + b - a

= b - a

= 20202019 - 20192019

= 10.000

Jawaban:

1. MS DOS

Kelebihan dari sistem DOS:

1. Memiliki ukurannya yang kecil.

2. Penggunaan memori juga cukup ringan

3. Dengan berbagai perangkat pun sangat kompatibel.

Kekurangan sistem DOS adalah hanya berupa tulisan serta tidak ada grafiknya, karena memang generasi pertama.

2. Microsoft windows

Kelebihan dari windows:

-Windows juga sudah user friendly sebab telah memiliki tampilan anta muka.

-Dalam proses instalasinya jauh lebih mudah.

-Windows juga sangat support driver.

-Banyak sekali perusahaan yang membuat software maupun aplikasi yang mana kompatibel dengan windows, karena memang hampir semua orang memakai sistem operasi windows.

-Selain itu, windows memiliki perkembangan yang sangat cepat, kita bisa melihat hampir pada setiap tahunnya akan selalu ada sebuah perubahan dari windows tersebut.

Kekurangan dari sistem windows:

-Windows tidak memiliki open source.

-Perlu untuk membeli sebab memang close source.

-Sangat mudah untuk di retas karena sistem keamanannya masih cukup lemah.

-Perlu membeli aplikasi rentan terkena virus terlebih dahulu, apabila tidak mempunyai software bawaan.

3. Linux

-

Kelebihan pada Linux:

-Karena memang open source, tidak akan ada biaya lisensi untuk pembelian dan menggunakannya.

-Jika dahulu penggunaan linux sangat susah, tapi setelah dilakukan pengembangan menjadi jauh lebih mudah.

-Sebagian aplikasi yang ada di windows juga ada di linux.

-Perkembangannya jauh lebih cepat sebab dikembangkan oleh orang banyak.

Kekurangan pada Linux:

-Perlunya diadakan sebuah pelatihan untuk pengguna awal sehingga dapat terbiasa dengan linux.

-Tidak semua support hardware yang cocok dengan linux.

-Proses instalasinya cukup susah apalagi untuk softwarenya.

Mac OS adalah sistem operasi yang dibuat oleh Apple. Sistem operasi ini sudah terpasang pada semua komputer Macintosh (disebut Mac) baru. Banyak orang lebih menyukai tampilan sistem operasi Mac OS X daripada Windows walaupun harga komputer Apple cenderung lebih mahal.

Sistem operasi merupakan sebuah aplikasi yang menggerakan harddisk atau laptop dan PC nya. Sistem operasi sangat penting dikarenakan tanpa adanya harddisk, kita akan susah menyimpan banyak file. Pengguna akan kesusahan dalam mengoperasikan hardware.

Pembahasan:

Secara umum, sistem operasi adalah perangkat lunak  lapisan pertama yang ditempatkan di memori komputer  saat komputer melakukan booting. Sistem operasi menjalankan layanan inti dari perangkat lunak, sementara perangkat lunak lain berjalan setelah sistem operasi berjalan. Sebelum pengenalan sistem operasi, komputer hanya menggunakan  sinyal analog dan  digital. Dengan berkembangnya ilmu pengetahuan dan teknologi, kini hadir berbagai sistem operasi dengan keunggulannya masing-masing.  Sistem operasi memiliki jadwal sistematis yang  mencakup  penggunaan memori, pemrosesan data, penyimpanan data, dan perhitungan sumber daya lainnya. Contoh sistem operasi modern termasuk Linux, Android, iOS, Mac OS X, dan Microsoft Windows.  Sistem operasi memainkan peran penting  dalam  sistem komputer. Berikut adalah beberapa fitur sistem operasi:

Manajemen sumber daya komputer

Sistem operasi dapat mengatur waktu eksekusi aplikasi, berbagi penggunaan CPU saat aplikasi berjalan pada waktu yang sama, dan mengizinkan akses disk. Bertindak sebagai aplikasi dasar untuk perangkat

Sistem operasi merupakan landasan untuk membentuk program-program yang ada pada  perangkat. Ini adalah bagian penting dalam mengelola semua yang Anda butuhkan untuk menjalankan fungsi perangkat Anda.

Perangkat keras koneksi

Sistem operasi bertanggung jawab untuk mengoordinasikan semua perangkat yang  terhubung secara bersamaan. B. Memori internal, mouse, speaker,  CPU.

Dalam hal ini, sistem operasi bertindak sebagai jembatan yang menghubungkan perangkat keras ke perangkat lunak. Kemudian  operasi komputer dasar dilakukan lagi.

Optimalisasi fitur perangkat

Sistem operasi dapat mengoptimalkan kinerja  perangkat keras dan perangkat lunak, dan sistem mengoordinasikan dan mengontrol hubungan perangkat keras-perangkat lunak sehingga dapat bekerja sama dengan baik.

Menyesuaikan sistem operasi perangkat

Sistem operasi mengoordinasikan dan mengontrol semua fungsi perangkat keras yang digunakan, seperti CPU, hard disk, dan memori. Tentu saja, sistem operasi memungkinkan semua perangkat untuk saling bersinergi dan membentuk unit untuk memaksimalkan fungsionalitas perangkat.

33. soal esay sistem operasi beserta jawabannya​


Jawaban:

1. Apa yang dimaksud sistem operasi komputer ?

Jawab:

Sistem Operasi Komputer adalah perangkat lunak komputer yang bertugas untuk melakukan kontrol dan manajemen perangkat keras dan juga operasi-operasi dasar sistem.

 

2. Secara kata dan dan istilah apa itu komputer? Jelaskan!

Jawab:

Suatu alat elektronik yang mampu melakukan tugas yaitu menerima memproses input sesuai dengan intruksi yang diberikan.

 

3. Secara umum  komponen sistem komputer terdiri dari 4 komponen, sebutkan !

Jawab:

Perangat keras Sistem operasi Program aplikasi User 

 

4. Jelaskan apa yang dimaksud perangkat keras !

Jawab:  

Perangkat keras  merupakan sumber daya utama untuk proses komputasi, contohnya CPU.

 

5. Jelaskan apa yang dimaksud user !

Jawab: 

User adalah yang menggunakan sistem.

 

6.  Jelaskan apa yang dimaksud dengan program apikasi !

Jawab: 

Program aplikasi adalah menentukan cara sumber daya sistem yang digunakan untuk menyelesaikan masalah.

 

7. Jelaskan secara singkat yang dimaksud dengan linux !

Jawab: 

Linux adalah software sistem operasi open source yang gratis untuk disebarluaskan di bawah lisensi GNU. Linux merupakan turunan dari unix dan dapat bekerja pada berbagai macam perangkat keras koputer mulai dari inter x86 sampai dengan RISC.

 

8. Sebutkan beberapa kelabihan dari sistem operasi linux !

Jawab:

Mudah didapat dan gratis, mudah digunakan, bebas dari virus. Aplikasi-aplikasinya dapat diperoleh dengan mudah dan gratis.  

 

9. Apa yang kamu ketahui tentang sistem komputer ?

Jawab:

Kumpulan elemen yang saling terhubung dan membentuk suatu kesatuan untuk melakukan tujuan pokok dari sistem tersebut.

 

10. Sebutkan dan jelaskan langkah-langkah yang dilakukan oleh komputer saat pertama kali dinyalakan !

Jawab:

Langkah pertama yang dilakukan oleh komputer saat pertama kali dinyalakan dalah bios memeriksa boot sector untuk boot record di ROM selanjutnya bios mengambil OS loader di dalam RAM kemudian  OS loader memeriksa sysetem file dan OS loader memindahkan OS ke RAM.

b. x² + 21x = 72

   x² + 21x - 72 = 0 ---> a = 1, b = 21, c = -72

   x₁₂ = [-b ± √(b² - 4ac)] / 2a

   x₁₂ = [-21 ± √((21)² - 4(1)(72))] / 2(1)

        = [-21 ± √153] / 2

        = [-21 ± 3√17] / 2

   x = (-21 + 3√17)/2     atau x = (-21 - 3√17)/2
