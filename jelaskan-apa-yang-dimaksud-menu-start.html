6. Tuliskan tuliskan dan jelaskan ikon-ikon yang terdapat pada sub menu start mailing merge yang ada pada menu mailing bersama dengan fungsinya​


Jawaban:

Fungsi dari tab Mailings adalah untuk membuat sebuah surat masal dengan format tulisan yang sama, akan tetapi mempunyai data yang berbeda dari setiap surat. Misalnya untuk pembuatan surat undangan dengan jumlah 100 undangan dengan format yang sama, akan tetapi harus memasukan nama dari penerima surat yang berbeda-beda.

Didalam tab mailings terdapat berbagi group/icon, diantaranya adalah group create,start mail merge, write&insert fields, preview results, finish. Untuk lebih jelasnya berikut penjelasan dari masing-masing group tersebut :

Enfelopes, adalan untuk membuat dan mencetak sampul pada amplop.

Labels, adalah untuk menuliskan alamat dan nama.

Start Mail Marge, adalah untuk membuat surat.

Select Resipiets, adalah untuk memilih penerima surat.

Edit Resipiet List, adalah untuk mengatur penerima surat.

Hailight Marge Fields aalah untuk menyisipkan sebuah dokument.

Address Block, adalah untuk memilih surat.

Greeting Line, adalah untuk menambah baris ucapan.

Insret Marge Field, adalah untuk menyisipkan dan menggabungkan surat.

Rules, adalah untuk memberikan karis bawah pada surat.

Match Filds adalah untuk membandingkan surat.

Update Labels, adalah untuk memperbarui lebel surat.

Preview Resuts, adalah untuk melihat hasil.

Find Resifient, adalah untuk mencari penerima.

Auto Check for Error, adalah untuk memeriksa kesalahan secara otomatis.

Finish&merge, adalah untuk mengetahui bahwa surat yang ada buat sudah lengkap dan bisa langsung dicetak secara otomatis.

11. tuliskan dan jelaskan fungsi dari elemen-elemen yang terdapat pada start menu windows 7!


Jawaban:

Desktop

Merupakan suatu tampilan awal pada saat Windows selesai di booting(tampilan home saat memasuki windows.

Wallpaper

Gambar yang menempel diatas desktop. Windows menyediakan berbagai macam wallpaper bawaan sehingga kita dapat mengubahnya dengan mudah dan cepat.

Start Menu

Tombol Start yang berisi berbagai macam program yang telah kita install, menu-menu lain yang dapat diakses oleh pengguna(user), seperti misalnya mendengarkan musik, menjalankan program editor teks dan lain-lain.

Icon

Gambar-gambar kecil yang menempel diatas desktop yang diikuti dengan identitasnya. Tujuan dari icon ini adalah untuk memudahkan dalam mengaktifkan suatu program aplikasi. Icon ini dapat ditambahi ataupun dikurangi sesuai dengan kebutuhan kita. Kalau terlalu banyak icon pada desktop akan mengurangi keindahan dari desktop itu sendiri.

Task Bar

Batang panjang yang terdapat paling bawah desktop. Kita dapat memindahkan taskbar ke atas atau samping kanan ataupun kiri desktop sesuai dengan keinginan kita sendiri. Pada taskbar ini terdapat program yang kita minimize-kan, icon-icon lainnya seperti icon pengaturan volume dan jam yang aktif.

Quick Launch

Bagian bawah sebelah kiri , sebelah kanan Start yang berisi icon-icon untuk shortcut program yang telah kita definisikan. Shortcut adalah cara cepat untuk menjalankan prgoram/data yang disimpan di suatu direktori tertentu. Shortcut berisi path menuju ke data/program yang dimaksud dan disimbolkan oleh icon tertentu.

12. jelaskan langkah langkah membuka microscoft word 2007 melalui menu start?


Langkah - langkah membuka Microsoft Word melalui menu start dengan :

Klik simbol Windows yang berada pojok kiri bawah ( sisi paling ujung pada taskbar )Jika program sudah di pin, maka akan muncul dalam pilihan depan saat simbol windows diklik. Jika belum, gunakan fitur search atau all programs untuk mencari. Microsoft Word jika dicari dalam all programs pada start, akan berada pada folder Microsoft Office.Klik Microsoft Word, tunggu beberapa saat maka program siap dijalankan dan digunakanPembahasan

Microsoft Word merupakan software pengolah angka yang hampir ada pada setiap perangkat karena beragam manfaatnya. Terkadang, Microsoft Word tidak selalu muncul shortcutnya pada desktop maupun taskbar sehingga perlu dicari melalui start. Akan tetapi, seiring berjalannya waktu dan perkembangan windows, mencari melalui start memiliki beberapa perbedaan.

Pada windows 7 dan versi sebelumnya, setelah menekan menu start maka akan terdapat pilihan program yang sudah dipin dan all program. Jika belum ada dapat digunakan menu all program -> Microsoft Office -> Microsoft Word. Untuk memudahkan kedepannya, bisa klik kanan -> pin to start sehingga ketika menekan menu start, Microsoft Word akan muncul dihalaman utama.

Pada windows 8, setelah menekan menu start, akan menuju ke tampilan beberapa menu - menu lain. Cari simbol panah ke kanan, lalu cari section Microsoft Office -> Microsoft Word 2007

Pada windows 10, setelah menekan menu start, akan tersedia nama - nama program dan foldernya yang sudah urut menurut abjadnya. Untuk memudahkan, setelah menemukan letak Microsoft Word dapat digunakan opsi pin to start sehingga akan muncul pada pilihan paling atas.
