Cara melakukan senam irama :

( langkah kaki biasa )

1. badan berdiri tegak.

2. Langkah kaki kiri ke depan sedangkan kedua tangan di samping badan.

3. Langkah kaki kanan maju ke depan dan jatuhkan ke tumit.

4. kaki kanan dan kiri melangkah secara bergantian.

( langkah rapat )

1. Berdiri tegak rileks.

2. Langkahkan kaki kanan di depan kaki kiri.

3. melangkahkan kaki kiri di depan kaki kanan.

4. Lanjutkan kedua kaki rapat.

5. Langkah keseimbangan.

( langkah keseimbangan )

1. Berdiri dengan sikap tegak rileks.

Hitungan 1: langkahkan kaki kiri ke depan.

Hitungan 2: kaki kanan menyusul melangkah ke depan.

2. Ketika tumit kaki kanan terangkat.

3. kaki kiri mundur diikuti kaki kanan mundur merapat lalu langkah ke depan dan selesai.

( langkah ke depan )

1. Berdiri dengan sikap anjur kiri.

Hitungan 1: silangkan kaki kiri di muka kaki kanan.

Hitungan 2: kaki kiri menyusul dan kaki kanan menyusul lagi.

2. langkahkan kaki kiri, disusul kanan, kemudian diikuti langkah kiri.

3. Langkah silang.

( langkah silang )

1. Berdiri dengan sikap anjur kiri.

Hitungan 1: silangkan kaki kiri di muka kaki kanan.

2. dapat dilakukan ke belakang.

3. Langkah silang ini dilakukan dengan irama 2/4.

( langkah tiga )

1. Awali dengan meletakkan kaki kanan melebar ke depan.

2. Lalu kaki kiri buka setengah langkah dengan menggunakan tumis.

3. Lakukan bergantian seperti itu antara kaki kanan dan kiri menggunakan posisi serupa.

( langkah lingkar )

3. Lakukan bergantian setelah meletakkan kaki kanan di tempat yang semula.
