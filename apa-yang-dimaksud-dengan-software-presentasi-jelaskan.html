Jawaban:

Perangkat lunak presentasi yaitu sebuah software/perangkat lunak komputer yang digunakan untuk membuat file-file presentasi.

Karena presentasi memiliki fungsi untuk mengutarakan sebuah gagasan-gagasan atau ide-ide, maka alat yang digunakan untuk presentasi haruslah memiliki tampilan yang menarik.

Software/aplikasi pembuat file presentasi dapat menyatukan objek yang unik seperti foto, video, dan suara.

Fungsi Perangkat Lunak Presentasi

Dari segi pengertian perangkat lunak presentasi, sebenarnya sudah jelas bahwa fungsi dasar dari software ini adalah untuk membuat file presentasi.

Namun, selain fungsi dasar tersebut software presentasi memiliki fungsi antara lain:

Membuat slide presentasi untuk mengajar, berbisnis, dan seminar

Menambahkan fitur-fitur seperti audio, video, gambar dan animasi dalam presentasi agar presentasi menjadi lebih menarik dan hidup

Mempermudah presenter dalam mengatur dan mencetak slide

Membuat presentasi dalam bentuk softcocopy sehingga dapat disimpan dan ditampilkan pada layar monitor.

Jenis-jenis Perangkat Lunak Presentasi

Sudah sangat banyak vendor-vendor teknologi membangun software yang berfungsi untuk membuat slide presentasi.

Namun, tiap-tiap software yang dikembangkan pasti memiliki keunggulan dan kekurangan masing-masing.

Berikut macam-macam software presentasi beserta kelebihan dan kekurangannya.

1. Power Point

￼

Power Point adalah software presentasi yang dikembangkan oleh perusahaan teknologi raksasa yaitu microsoft.

Software ini adalah software presentasi yang paling terkenal terutama di Indonesia.

Microsoft Power point juga merupakan salah satu pioneer perkembangan software presentasi yang canggih dan menarik.

Kelebihan:

Penyajian Power point sangat menarik karena warna, huruf dan animasi, dapat dimainkan sesuai selera

Easy user, Informasi dijelaskan secara visual sehingga mudah dipahami peserta didik

Dapat diperbanyak sesuai kebutuhan

Dapat dipakai secara berulang kali.

Kekurangan:

Karena memiliki banyak versi, Format file versi terbaru tidak dapat di edit di versi sebelumnya, kecuali apabila format file diganti terlebih dahulu seperti file terdahulunya.

Harga software yang cukup mahal. Di Indonesia, software ini sangat terkenal meskipun ditaksir dengan harga mahal. Hal itu karena masyarakat Indonesia banyak yang menggunakan software bajakan

2. iWork Keynote

￼

Software ini merupakan software bawaan dari perusahaan teknologi pesaing microsoft yaitu Apple. Software ini hadir dihampir seluruh produk Mac.

Seperti produk-produk Apple yang lain, software ini terkenal dengan tampilannya yang ciamik.

Terlihat sederhana tetapi sangat elegan. Software ini juga memiliki grafis yang sangat baik dengan fitur yang lebih banyak.

Kelebihan dan Kekurangan Keynote antara lain:

Kelebihan:

Covernya sangat berbeda, memukau, dan mudah digunakan

Meskipun merupakan produk dari Apple, software ini men-support file-file dari microsoft office

Efek dan transisi yang ditawarkan lebih baik dan memiliki fitur yang lebih lengkap daripada power point.

Kekurangan:

Format yang dihasilkan hanya bisa dibaca oleh produk-produk Apple.

Format keynote ini bisa diubah ke file dengan format .ppt milik power point. Namun, segala efek dan transisi dramatis pada Keynote akan hilang.

Karena merupakan produk bawaan dari Apple, maka untuk menggunakannya kalian harus membeli Mac yang tentunya memiliki harga yang tidak murah.

3. Libre Office Impress

￼

Libre office merupakan software presentasi yang dikembangkan oleh SUN Microsystem dengan bahasa Java.

Software ini telah mendukung cross platform yang berarti dapat dijalankan di pelbagai OS (Operation System).

Software ini cukup terkenal karena merupakan software presentasi yang gratis namun fitur-fiturnya tidak kalah dengan yang lain. Dengan fitur yang tak kalah melimpah dibanding Microsoft Office.

Kelebihan:

Merupakan Software Open Source, atau software gratisan

Kompatibel dengan Microsoft Office dan format-format lainnya

Ekspor format ke PDF atau format lain mudah

Tampilannya menarik dan mudah digunakan.

Kekurangan:

“Rakus” dengan daya komputer, perlu komputer RAM besar untuk menampungnya

Seperti Keynote, Format yang dihasilkan bisa di ubah ke format .ppt. Namun, tampilannya jadi berantakan.

4. Prezi

￼

Prezi merupakan sebuah software presentasi yang bisa melihat dan mengedit file presentasi dari perangkat apapun.

Semua hasil presentasi yang dibuat akan tersimpan di Cloud. Tetapi juga bisa ditelusuri secara offline.

Prezi juga memberikan layanan alat editing gambar yang cukup keren. File presentasi dari prezi sangat mudah jika dibagikan dengan orang lain.

Kelebihan:

Tampilan tema yang lebih meningkatkan performa dibandingkan dengan power point

Menggunakan teknologi ZUI, sehingga berbeda dengan power point atau yang lain

Tampilannya dan animasinya simple

Template yang bervariasi yang dapat di unduh secara online.

Kekurangan:

Instalasi Prezi membutuhkan jaringan komputer yang stabil

Sulit menginput simbol-simbol matematika

Berbayar, meskipun tidak semahal power point.
