2. Asam Sulfat direaksikan dengan Natrium Hidroskida menghasilkan senyawa garam Natrium Sulfat dan Air . Jika Asam Sulfat yang digunakan tersebut sebanyak 24,5 gram . (Ar H = 1 : S = 32 : O = 16 dan Ar Na = 23) Buktikanlah Hukum Lavosier dan Hipotesa Avogadro .serta Hk Perbandingan Gay-Lussac ( Jika semua zat adalah Gas pada STP ) tolongg dijawab yang tau dikumpul besok soalnya #ngasalreport


Penjelasan:

H2SO4 + 2 NaOH => Na2SO4 + 2 H2O

Mr H2SO4 = ( 2 x Ar H ) + ( 1 x Ar S ) + ( 4 x Ar O )

=> 2 + 32 + 64

=> 98

Mr NaOH = ( 1 x Ar Na ) + ( 1 x Ar O) + ( 1 x Ar H )

=> 23 + 16 + 1

=> 40

Mr Na2SO4 = ( 2 x Ar Na ) + ( 1 x Ar S ) + ( 4 x Ar O )

=> 46 + 32 + 64

=> 142

Mr H2O = ( 2 x Ar H ) + ( 1 x Ar O )

=> 2 + 16

=> 18

mol H2SO4 = massa / Mr

=> 24,5 / 98

=> 0,25 mol

Perbandingan koefisien ~ perbandingan mol ~ perbandingan volume

Koefisien H2SO4 : NaOH : Na2SO4 : H2O

=> 1 : 2 : 1 : 2 berarti mol masing adalah

=> 0,25 : 0,5 : 0,25 : 0,5

massa NaOH = n x Mr

=> 0,5 x 40

=> 20 gram

massa Na2SO4 = n x Mr

=> 0,25 x 142

=> 35,5 gram

massa H2O = n x Mr

=> 0,5 x 18

=> 9 gram

Hukum kekekalan massa => massa zat sebelum reaksi = massa zat sesudah reaksi

massa H2SO4 + massa NaOH = massa Na2SO4 + massa H2O

=> 24,5 + 20 = 35,5 + 9

=> 44,5 gram ( massa sebelum reaksi )

=> 44,5 gram ( massa sesudah reaksi )

Berarti sesuai hukum kekekalan massa ✔️✔️✔️
