2)Modus ponens

dalam menarik kesimpulan di logika

4)Modus tollens dalam menarik kesimpulan di logika

5)Modus Silogisme

Modus silogisme ditandai dengan adanya dua pernyataan majemuk implikasi

Berikut benda yang dapat ditarik dan tidak dapat ditarik oleh magnet:

1. Pensil : di tolak, lemah

2. Pulpen : di tolak, lemah

3. Mistar plastik : di tolak, lemah

4. Paku besi : di tarik, kuat

5. Sendok aluminium : di tarik, lemah

6. Timah : di tolak, lemah

Penjelasan:

Magnet merupakan seluruh bahan atau benda yang memiliki kemampuan untuk menarik berbagai jenis logam tertentu. Tidak semua jenis logam dapat ditarik oleh magnet. Maka dari itu, benda-benda terbagi menjadi 3 berdasarkan daya tariknya terhadap magnet, yaitu feromagnetik, paramagnetik, dan diamagnetik.

Feromagnetik

Feromagnetik adalah kategori benda yang dapat ditarik dengan kuat oleh magnet. Benda yang tergolong feromagnetik dapat diubah menjadi magnet. Contoh: besi, baja, dan kobalt.

Paramagnetik

Paramagnetik adalah kategori benda yang ditarik lemah oleh magnet. Contoh: alumiunium, platina, magnesium, dan titanium.

Diamagnetik

Diamagnetik adalah kategori benda yang tidak dapat ditarik oleh magnet, bahkan ditolak dengan leah oleh magnet. Contoh: emas, timah, dan bismuth.

26. Contoh soal penarikan kesimpulan yg sah dalam modus silogisme


Contoh soal : modus silogisme 1. Tentukan kesimpulan dari premis berikut ini :) Jika x bilangan real, maka x2 ≥ 0 Jika x2 ≥ 0, maka (x2 + 2) > 0 Pembahasan : Untuk mempermudah, lakukan pemisalan sebagai berikut : » x bilangan real = A » x2 ≥ 0 = B » (x2 + 2) > 0 = C Dengan menggunakan kaidah silogisme, maka pernyataan di atas dapat disusun menjadi : A ⇒ B B ⇒ C ∴ A ⇒ C Dengan demikian, kesimpulan dari premis di atas adalah : Jika x bilangan real, maka (x2 + 2) > 0. 2. Diketahui pernyataan sebagai berikut : Jika Rihanna konser di Jakarta, maka saya akan menonton Jika saya menonton, maka saya sangat senang Tentukan kesimpulan yang sah tentang pernyataan di atas. Pembahasan : Rihanna konser di Jakarta : P Saya menonton : Q Saya sangat senang : R Dengan menggunakan silogisme, maka pernyataan di atas dapat disusun menjadi : P ⇒ Q Q ⇒ R ∴ P ⇒ R Jadi, kesimpulan dari pernyataan di atas adalah : Jika Rihanna konser di Jakarta, maka saya sangat senang.

34. Tolong dong ka ajarin logika matematika yang menarik kesimpulan...


jadi logika matematika ( penarikan kesimpulan ) itu ada 3 : 

1.  Modus Ponens 
      premis 1 : p --> q 
      premis 2 : q 
____________________
kesimpulannya q 
      
      contoh 
      premis 1 : ibu beli buah mangga adik merasa senang 
      premis 2 : ibu beli mangga 
______________________________________________
kesimpulannya : adik merasa senang 
*jika premis kedua ibu beli alpukat maka kesimpulannya adik akan merasa sedih 

2. Modus Tollens
    premis 1 : p --> q 
    premis 2 : ~p
