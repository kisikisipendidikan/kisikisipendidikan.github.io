1. aturan sinus, aturan kosinus,dan rumus luas segitiga


Aturan Sinus, Aturan Cosinus, Luas segitiga ?

[1] Aturan Sinus

Sin A / a = Sin B / b = Sin C / c

Dapat digunakan saat mencari salah satu sisi segitiga yang diketahui kedua sudutnya dan salah satu sisinya

[2] Aturan Cosinus

a^2 = b^2 + c^2 - 2bc cos A
b^2 = a^2 + c^2 - 2ac Cos B
c^2 = a^2 + b^2 - 2ab Cos C

Dapat digunakan untuk mencari sisi salah satu segitiga yang diketahui kedua sisinya dan sudut sisi yang dicari

[3] Luas Segitiga

L = 1/2 a x t

L = 1/2 a b Sin C
L = 1/2 b c Sin A
L = 1/2 a c Sin B

L = akar s(s-a)(s-b)(s-c)
s = 1/2 k

sudut A = 120°
sudut C = 30°
sudut B = 180° - (120°+30°) = 30°

sudut B = sudut C (∆ABC sama kaki)

c = AB = 6
b = AC
a = BC

aturan sinus
AB/sin C = AC/sin B
6/sin 30° = AC/sin 30°
AC = 6

aturan cosinus
BC² = AB² + AC² - 2.AB.AC.cos A
BC² = 6² + 6² - 2.6.6.cos 120°
BC² = 36 + 36 - 72.(-½)
BC² = 72 + 36
BC² = 108
BC = √108
BC = 6√3

RUMUS JUMLAH DAN SELISIH SINUS DAN KOSINUS

Rumus jumlah dan selisih sinus dan kosinus merupakan bentuk manipulasi dari rumus hasil kali sinus dan kosinus yang telah dibahas sebelumnya. Rumus-rumusnya adalah sebagai berikut.

sin A + sin B = 2.sin ½(A + B). cos ½(A – B)

sin A − sin B = 2.cos ½(A + B). sin ½(A – B)

cos A + cos B = 2.cos ½(A + B). cos ½(A – B)

32. Bagaimana cara menggunakan rumus kosinus dalam menentukan besar vektor resultan


Untuk hal tersebut dapat menggunakan rumus :

[tex]R = \sqrt{ a^{2} + b^{2} + 2. a.b. cos \alpha } [/tex]

Sedangkan untuk selisih vektor menggunakan rumus :

[tex]R = \sqrt{ a^{2} + b^{2} - 2. a.b. cos \alpha } [/tex]
untuk mencari resultan vektor pada kosinus  , bisa di gunakan dengan cara : 

R = √x² + y² + 2xy. cos α
 kalau untuk mencari selisih nya tinggal menggantikan ( + ) menjadi tanda ( -)

37. aturan sinus kosinus dan luas segitiga


Aturan sinus :
        Jika ada segitiga sembarang ABC maka berlaku :
        BC/sin∠A = AC/sin∠B = AB/sin∠C
Aturan cosinus :
        Jika ada segitiga sembarang ABC maka berlaku :
        ( i )   BC^2 = AC^2 + AB^2 - 2 AC . AB cos∠A
        ( ii )  AC^2 = BC^2 + AB^2 - 2 BC  . AB cos∠B
        ( iii ) AB^2 = BC^2 + AC^2 - 2 BC . AC cos∠C
Luas Segitiga :
        Jika ada segitiga sembarang ABC maka berlaku :
        ( i )   luas Δ ABC = 1/2 AC . AB sin∠A
        ( ii )   luas Δ ABC = 1/2 BC . AB sin∠B
        ( iii )  luas Δ ABC = 1/2 BC . AC sin∠C
