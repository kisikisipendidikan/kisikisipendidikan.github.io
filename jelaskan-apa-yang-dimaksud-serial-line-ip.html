4. buat perintah command line untuk ip address LAN-1 dengan 192.168.100.12/24​


JAWABAN

Mengkonfigurasi pengalamatan jaringan (ip address) pada sistem operasi windows adalah hal yang sangat mudah, mungkin kawan-kawan semua setuju. Tampilan windows yang berbasis grafik (GUI - Graphical User Interface) sangat memudahkan user untuk mengoperasikannya. Kita cukup menekan icon-icon dan menu menggunakan mouse atau touchpad.

Konfigurasi IP Melalui Command Prompt

Untuk mengkonfigurasi ip address pada sistem operasi windows, biasanya kita akan menuju ke Control Panel > Network and Internet > Network and Sharing Center atau dengan mengeklik kanan icon network yang ada pada taskbar kemudian memilih opsi Open Network and Sharing Center lalu memilih change adapter settings, Kemudian mengkonfigurasi adapter jaringannya.

Bagaimana jika konfigurasi ip dilakukan menggunakan command line seperti saat kita mengkonfigurasi ip di sistem operasi linux berbasis teks? Tentu akan terasa menarik dan mungkin saja proses konfigurasinya akan terasa lebih simple dan cepat.

Di windows kita dapat menggunakan command prompt untuk mengkonfigurasi pengalamatan jaringan (ip address, gateway, dns). Untuk melakukan hal tersebut kita bisa menggunakan perintah netsh yang ada pada command prompt.

Netsh (Network Shell) merupakan utility perintah yang digunakan untuk memodifikasi konfigurasi jaringan seperti alamat ip, subnet prefiks, gateway serta server dns. Bahkan perintah netsh juga bisa digunakan untuk mengkonfigurasi firewall.

Pada postingan ini kita hanya fokus membahas tentang perintah-perintah netsh yang digunakan untuk melakukan konfigurasi pengalamatan jaringan pada komputer. Berikut adalah beberapa perintahnya :

Melihat daftar interface yang ada pada komputer

netsh interface show interface

Perintah tersebut akan menampilkan daftar interface jaringan yang ada pada komputer. Contoh :

Melihat konfigurasi ip

netsh interface ipv4 show config

Perintah di atas akan menampilkan info konfigurasi pengalamatan ip versi 4 pada setiap interface yang ada pada komputer, seperti ip address,  gateway, serta dns server.

Kita bisa juga menggunakan perintah berikut :

netsh interface ipv4 show addresses

Bedanya dari perintah sebelumnya adalah, tidak ada informasi tentang dns server dan wins server yang ditampilkan. Perintah tersebut hanya menampilkan info konfigurasi ip address, subnet mask, gateway, serta metric.

5. Bila seseorang membutuhkan sebuah IP Public untuk digunakan pada sebuah server yang akan ditempatkan on-line 24 jam di internet, maka dari mana orang tersebut dapat memperoleh IP Public yang diinginkan?


Jawaban:

seseorang harus menghubungi penyedia layanan internet (Internet Service Provider atau ISP).

Penjelasan:

Untuk memperoleh sebuah IP Public yang diinginkan untuk digunakan pada server yang akan ditempatkan secara online 24 jam di internet, seseorang harus menghubungi penyedia layanan internet (Internet Service Provider atau ISP). ISP adalah perusahaan yang menyediakan akses internet kepada pengguna, baik individu maupun organisasi.

Berikut adalah langkah-langkah umum yang dapat diambil untuk memperoleh IP Public:

Pilih ISP: Tentukan ISP yang akan Anda gunakan untuk menyediakan koneksi internet. Pilihlah ISP yang memiliki reputasi baik dan menawarkan layanan yang sesuai dengan kebutuhan Anda.Hubungi ISP: Hubungi tim dukungan teknis atau penjualan ISP tersebut. Laporkan kebutuhan Anda untuk mendapatkan IP Public untuk server yang akan ditempatkan secara online. Anda dapat menjelaskan bahwa server tersebut memerlukan alamat IP Public yang tetap dan stabil.Verifikasi kebutuhan: ISP akan melakukan verifikasi kebutuhan Anda dan memberikan informasi lebih lanjut tentang langkah-langkah yang perlu Anda ikuti. Mereka mungkin meminta Anda untuk mengisi formulir aplikasi atau memberikan informasi tambahan terkait server dan kebutuhan Anda.Konfigurasi dan pengaturan: Setelah permohonan Anda disetujui, ISP akan mengkonfigurasi jaringan mereka untuk memberikan IP Public yang diminta kepada server Anda. Mereka mungkin memberikan instruksi tentang pengaturan dan konfigurasi yang perlu Anda terapkan pada server Anda.Pelaksanaan: Setelah konfigurasi selesai, Anda akan diberikan IP Public yang dapat digunakan untuk mengakses server secara online 24 jam di internet.

Perlu diingat bahwa penyediaan IP Public mungkin melibatkan biaya tambahan atau memerlukan paket layanan khusus dari ISP. Pastikan untuk membahas hal ini dengan ISP yang Anda pilih agar Anda memahami persyaratan dan biaya yang terkait dengan mendapatkan IP Public.

26. jelaskan yang di maksud dengan IP dinamik dan IP statik?


IP Static dan IP Dynamic adalah ptotokol atau tatacara mendapatkan alamat host. Static menunjukkan bahwa "pemberian" alamat IP adalah tetap dan tidak akan didistribusikan ke host lain walau host yg dituju tidak ada. 
Dynamic menunjukkan bahwa "pemberian" alamat IP adalah berubah dan akan didistribusikan ke host lain jika host dengan alamat yg sama sudah tidak ada lagi. 

IP statik= 1 IP untuk 1 modem, jadi kalau kamu pake provider yg pake IP statik maka dari awal pemasangan sampai akhir IPmu akan itu terus... 

IP dinamic= beberapa IP untuk beberapa modem, jadinya biasanya setiap kali konekin modem dia akan dapetnya berbeda beda, tergantung IP yg tersedia...

34. Jelaskan apa yang dimaksud dengan TCP IP Sebutkan dan jelaskan protokol apa saja yang ada di dalamnya


TCP/IP (Transmission Control Protocol/Internet Protocol)

Dalam Bahasa Indonesia disebut Protokol Kendali Transmisi/Protokol Internet.

Sebuah protokol atau rule atau aturan yang memiliki fungsi utama untuk saling bertukar informasi antar komputer melalui sebuah jaringan (Internet).

Protocolnya ada banyak. Ini beberapa contoh:
- FTP (File Transfer Protocol)
- Telnet
- NFS (Network File System)
- SMTP (Simple Main Transfer Protocol)
- POP
- Banyak lagi
