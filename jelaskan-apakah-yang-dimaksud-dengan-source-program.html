15. apa perbedaan program berlisensi freeware dan open source?​


Jawaban:

Free Software

Free Software adalah sebuah software yang menghargai kebebasan pengguna maupun masyarakat. Dalam hal ini, artinya pengguna memiliki kebebasan untuk menjalankan, mempelajari, mendistribusi, mengubah, memodifikasi, dan meningkatkan fungsi dan kemampuan software.

Kebebasan ini tercantum dalam 4 esensi kebebasan sebagai berikut :

Kebebasan untuk menjalankan program dalam tujuan apapun semau anda

Kebebasan untuk mempelajari cara kerja program, dan mengubahnya semau anda. Adanya akses ke sumber kode merupakan prasyarat dari kebebasan ini.

Kebebasan untuk mendistribusi salinan untuk membantu orang lain

Kebebasan untuk mendistribusi salinan yang telah dimodifikasi kepada orang lain. Dengan melakukan hal ini, anda memberikan kesempatan kepada seluruh masyarakat untuk merasakan manfaat dari perubahan program anda. Adanya akses ke sumber kode merupakan prasyarat dari kebebasan ini.

Contoh : GIMP, Ubuntu OS, Linux Kernel, Kdenlive, Vega Strike, Naev, dan banyak lagi.

Open Source

Open source adalah sebuah istilah dimana sebuah software yang memiliki kode sumber yang bisa diakses oleh semua orang secara terbuka. Biasanya ditandai dengan adanya sumber kode dan lisensi yang memperbolehkan user untuk mengembangkan, mengaplikasikan, dan memperbaiki kelemahannya. Biasanya istilah Open Source ini memfokuskan pada nilai-nilai teknis dalam pembuatan software yang memiliki nilai guna dan dapat dihandalkan.

Open Source memiliki kriteria sebagai berikut :

•Pendistribusian ulang secara bebas

•Kode Sumber

•Karya derivasi (turunan)

•Integritas kode sumber dari pencipta

•Tidak ada diskriminasi terhadap individu dan kelompok

•Tidak ada diskriminasi terhadap bidang pekerjaan

•Pendistribusian lisensi

•Lisensi tidak boleh bersifat spesifik pada produk

•Lisensi tidak boleh membatasi software lain

•Lisensi harus netral secara teknologi (Technology-Neutral)

---------------------------------------------------------

Penjelasan:

Freeware

Freeware merupakan sebuah program yang tersedia secara gratis. Program ini biasanya tersedia dengan gratis tapi tidak tersedianya kode sumber. Selain itu, adna tidak boleh menjualnya karena kendala EULAnya. Sehingga, program ini tidak bebas, alias nonfree.

Contoh : Free Download Manager, FormatFactory, BatteryCare, dsb.

Free Software dan Open Source pada dasarnya memiliki tujuan yang sama, namun yang membedakannya adalah filosofinya. Freeware dengan Free Software dan Open Source bedanya adalah boleh apa tidaknya menyebarkan, mendistribusi, ada atau tidaknya source code, maupun memodifikasi. Lisensi juga menentukan itu Free Software apa bukan.

Kesalahan pertama yang umum terjadi adalah mengartikan Free pada Free Software sebagai gratis. Free di sini artinya bebas, bukan gratis. Gunakan analogi bebas-tidak bebas (Free-nonfree), bukan gratis-berbayar (gratis-paid).

Jadi, analogikan kalau Free Market, ini artinya Pasar Bebas. Bukan pasar gratis.

Intinya :

Free : Free Software, Jawaban berlisensi bebas(Free License/Open License)

Nonfree/Proprietary : Freeware, jawaban yang dilabeli "Tidak untuk Direproduksi", Proprietary Software
