8. contoh soal masalah yang beekaitan dengan suhu beserta jawabanya


Suhu di Jambi adalah 23°C sedangkan suhu sebungkus eskrim di Surabaya adalah -8° C, berapa selisih suhu di Jambi dengan suhu eksrim di Surabaya ?

Jawab : selisih suhu keduanya adalah
= 23° C - (-8° C)
= 31° C ✔️

Semoga membantuMasalah tentang skala suhu. Suhu dibedakan menjadi 4 jenis, yaitu : Celcius ( C ), Reamur ( R ), Fahrenheit ( F ), dan Kelvin ( K ). Dari itu, kita dapat mengetahui perbandingannya sebagai berikut. Perbandingan = C : R : F : K = 5 : 4 : 9 : 5. Pembahasan tentang mengkonversikan skala Celcius menjadi skala lainnya, sebagai berikut. C menjadi R = 4/5 x suhu C. C menjadi F = ( 9/5 x suhu C ) + 32. C menjadi K = ( 5/5 x suhu C ) + 273,15.
