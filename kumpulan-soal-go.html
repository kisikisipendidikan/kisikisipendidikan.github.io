Penjelasan:

Conditional sentence merupakan kalimat pengandaian atau bersyarat. Conditional sentence terdiri dari if clause dan main clause. Conditional sentence terbagi menjadi 3, yaitu :

↪Conditional sentence type 1 [Future Possible]

digunakan pada masa sekarang dan menunjukkan sesuatu yang masih sangat mungkin terjadi. Rumus :

▶ If Simple present + Simple future.

contoh :

If I go to school by foot, I will be late.I will be hungry if I don't have breakfast.

↪Conditional Sentence type 2 [Present Unreal]

digunakan pada masa sekarang dan menunjukkan sesuatu yang tidak mungkin terjadi (berlawanan dengan fakta). Rumus :

▶If simple past + would simple form.

contoh :

If I were here, I would help you.I would buy this book if this book were needed.

↪Conditional Sentence type 3 [Past Unreal]

digunakan untuk menunjukkan sesuatu dimasa lampau dan tidak terjadi (berlawanan dengan fakta). Rumus :

▶ If past perfect + would have past participle

contoh :

If I had had enough money, I would have gone with you.I would have gone to zoo if the weather had been nice.
