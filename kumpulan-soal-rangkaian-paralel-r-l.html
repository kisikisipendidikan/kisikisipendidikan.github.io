Rp = 50 Ω

Penjelasan:

[tex]\frac{1}{r_{p} }[/tex] = [tex]\frac{1}{r_{1} }[/tex] + [tex]\frac{1}{r_{2} }[/tex] + [tex]\frac{1}{r_{3} }[/tex]

[tex]\frac{1}{r_{p} }[/tex] = [tex]\frac{1}{200}[/tex] + [tex]\frac{1}{200}[/tex] + [tex]\frac{1}{100}[/tex]

[tex]\frac{1}{r_{p} }[/tex] = [tex]\frac{1}{200}[/tex] + [tex]\frac{1}{200}[/tex] + [tex]\frac{2}{200}[/tex]

[tex]\frac{1}{r_{p} }[/tex] = [tex]\frac{4}{200}[/tex]

[tex]\frac{1}{r_{p} }[/tex] = [tex]\frac{1}{50}[/tex]

Rp = 50 Ω
