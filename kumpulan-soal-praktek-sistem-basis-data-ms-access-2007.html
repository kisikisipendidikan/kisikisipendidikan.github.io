30. yang tidak termasuk data type yang disediakan ms access adalah


Yang tidak termasuk data type yang disediakan microsof access adalah mari kita bahas terlebih dahulu data apa saja yang ada pada microsoft access. Namun kita akan bahas terlebih dahulu apa itu microsoft access. Microsoft Access adalah sebuah aplikasi buatan Microsoft yang berfungsi untuk database dalam skala kecil dan menengah.

Tipe data yang ada pada microsoft access adalah sebagai berikut

1. Text, tipe data text merupakan salah satu tipe data yang umum digunakan. Tipe data ini biasanya digunakan untuk data nama, alamat dan lainnya. Maksimal karakter adalah 255 karakter.

2. Memo, tipe data yang kedua pada microsoft access adalah memo. Memo tidak berbeda jauh dengan text, perbedaan mendasar adalah jumlah karakter, jika text hanya 255 karakter, memo bisa menampung hingga 64.000 karakter.

3. Number, tipe data ketiga adalah number, pada tipe data ini biasanya berguna untuk melakukan perhitungan. Seperti penambahan, pengurangan, pembagian dan sebagainya.

4. Date Time, tipe data selanjutnya pada microsoft access adalah tipe data date time. Tipe data ini berfungsi sebagai isian data waktu, seperti jam, tanggal dan sebagainya.

5. Currency, tipe data selanjutnya ada;ah currency. Tipe data ini sering digunakan sebagai data mata uang.

Demikian pembahasan mengenai tipe data pada microsoft access, untuk tipe data yang tidak terdapat pada microsoft access adalah unicode dan binary. Tipe data tersebut terdapat di SQL Server.

36. Tuliskan cara menghapus data yang tidak diperlukan MS Access
