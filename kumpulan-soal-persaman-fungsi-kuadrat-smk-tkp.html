21. kurva fungsi kuadrat memotong sb x dititik 4,0 dan 6,0 jika kurva melalui titik 3, -9 tentukan persamannya​


Untuk kurva y = ax² + bx + c dengan titik-titik (4,0) dan (6,0) yang melalui titik (3,-9), maka kita bisa menggunakan rumus persamaan kuadrat untuk menentukan a, b, dan c.

0 = a * 4^2 + b * 4 + c

0 = a * 6^2 + b * 6 + c

-9 = a * 3^2 + b * 3 + c

Menyelesaikan sistem persamaan ini menggunakan matriks atau metode lain, hasilnya adalah a = -3, b = 15, c = -27. Maka persamaan kuadrat adalah y = -3x² + 15x - 27.

34. kurva fungsi kuadrat memotong sumbu x dititik 4,0 dan 6,0 jika kurva melalui titik 3, -9 tentukan persamannya​


Dengan menggunakan teorema diferensial, kita tahu bahwa titik tengah dari dua titik (4, 0) dan (6, 0) adalah (5, 0). Dengan demikian, kita bisa menentukan bahwa koefisien kuadrat pada persamaan kurva adalah -9.

Dengan menambahkan titik (3, -9), kita bisa menentukan bahwa koefisien linier adalah 6.

Dengan demikian, persamaan kurva adalah:

y = -9(x - 5)^2 + 6x - 27.
