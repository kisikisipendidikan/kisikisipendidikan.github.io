1. Jelaskan instrumen dan mekanisme yang digunakan untuk melakukan kebijakan operating leverage dan financial leverage!​


Jawaban:

- Operating leverage

Operating Leverage dapat didefinisikan sebagai kemampuan perusahaan memanfaatkan biaya tetap untuk menghasilkan laba yang lebih baik bagi perusahaan. Operating Leverage adalah rasio biaya tetap terhadap biaya variabel. Jika perusahaan memiliki biaya tetap yang lebih tinggi dibandingkan dengan biaya variabel, maka perusahaan tersebut dikatakan memiliki Operating Leverage tinggi.

- Financial leverage

Financial Leverage mengacu pada jumlah utang dalam struktur modal perusahaan bisnis.Financial Leverage mengacu pada bagaimana perusahaan membayar aset tersebut untuk atau bagaimana aktivitas operasional akan dibiayai.

Penggunaan Financial Leverage dalam membiayai operasi perusahaan benar-benar dapat meningkatkan laba atas ekuitas dan laba per saham perusahaan. Ini karena perusahaan tidak merendahkan pendapatan pemilik (investor) dengan menggunakan pembiayaan dari ekuitas yang tinggi. Tingginya prosentase Financial Leverage dapat meningkatkan risiko kebangkrutan.


2. a. Hitunglah degree of operating leverage b. Apabila diasumsikan bahwa degree of operating leverage adalah 2,0. Hitunglah tingkat kenaikan laba yang akan terjadi apabila penjualan pada tahun yang akan datang diekspektasi akan meningkat sebesar 20 persen


a. 2,25

b. 62,5%

Pembahasan

Rumus Dasar :

DOL = Q (P – V) / Q ((P – V) – FC)

Keterangan

DOL = degree of operating leverageQ = Kuantitas produk yang terjualP = Harga per unitV = Variabel CostFC = Fixed Cost

Penjelasan

a.

Hitung besarnya Variabel Cost

= 10.000 + 4.000

= 14.000

Hitung besarnya fixed cost

= 12.000 + 8.000

= 20.000

maka,

DOL = Q (P – V) / Q ((P – V) – FC)

DOL = 4.000 (50.000 - 14.000) / 4.000 ((50.000 - 14.000) - 20.000

DOL = 4.000 (36.000) / 4.000 (36.000 - 20.000)

DOL = 144.000.000 /4.000 (16.000)

DOL = 144.000.000 / 64.000.000

DOL = 2,25

b.

Besar Laba Awal

= 50.000 - 10.000 - 12.000 - 4.000 - 8.000

= 50.000 - 34.000

= 16.000

Diperkirakan penjualan meningkat 20%, maka besar penjualan tahun itu adalah :

= 50.000 + (20% × 50.000)

= 50.000 + 10.000

= 60.000

Maka, labanya menjadi :

= 60.000 - 10.000 - 12.000 - 4.000 - 8.000

= 60.000 - 34.000

= 26.000

Tingkat kenaikan labanya :

= (26.000 - 16.000) / 16.000 × 100%

= 10.000/16.000 × 100%

= 0,625 × 100%

= 62,5%

Jadi, tingkat kenaikan labanya sebesar 62,5%

7. Jelaskan cara menginstal operating sistem


LANGKAH 1

Hubungkan monitor, papan kekunci, tetikus dan peranti-peranti lain pada PC. Jangan lupa sambungkan kabel power ke port power dibelakang PC. Sesudah itu, tekan buton power pada PC dan monitor untuk menghidupkan komputer rakitan Anda. PC akan segera melakukan proses booting.


LANGKAH 2

Segera menekan buton [Delete] pada papan kekunci untuk masuk ke menu BIOS/CMOS Setup.


LANGKAH 3

Anda akan masuk BIOS. Pada bahagian ini, motherboard yang digunakan akan menggunakan Award BIOS. Jadi langkah-langkah berikut ini disusun berdasarkan Award BIOS. Pengaturan untuk jenis BIOS lainnya lebih kurang sama sahaja. Hal pertama yang Anda harus lakukan adalah menetapkan waktu yang akan digunakan oleh sistem komputer. Tetapkanlah melalui menu [Main][System Time]. Untuk mengubahnya, gunakanlah buton [+], [-] dan buton[Tab] pada papan kekunci Anda


LANGKAH 4

Seterusnya dengan cara yang sama, tetapkanlah tarikh pada sistem komputer Anda melalui menu[System Date].


LANGKAH 5

Setelah menetapkan waktu dan tarikh dalam sistem, Anda juga dapat memasang password BIOS supaya orang lain tidak dapat mengubah BIOS yang telah Anda buat. Caranya,pilih menu [Supervisor Password], lalu tekan tombol [+] pada papan kekunci. Selanjutnya, masukkan password Anda pada bahagian Enter Password, lalu klik [Enter]. Masukkan kembali password Anda pada bahagian Confirm Password, kemudian kembali klik [Enter]. Sekarang status Supervisor Password pasti menjadi Enabled.


LANGKAH 6

Bila PC Anda digunakan secara beramai-ramai, Anda juga boleh menetapkan user password untuk setiap pengguna. Manfaatkan saja menu [User Password]. Cara menetapkannya sama sahaja seperti pengaturan pada supervisor password.


LANGKAH 7

Sekarang bukalah menu [Advanced] dengan menekan buton [‡] (arah panah ke kanan) pada papan kekunci Anda, lalu pilih [I/O Device Configuration] kemudian [Enter].


LANGKAH 8

Di sini Anda dapat mengatur penggunaan fitur on-board pada motherboard Anda. Misalnya, bila Anda menggunakan kartu suara dan modem yang bukan on-board , ubahlah [Onboard AC97Audio Controller] dan [Onboard AC97 Modem Controller] menjadi [Disabled] dengan menekan buton [+] pada papan kekunci Anda. Selanjutnya klik [Esc] untuk kembali ke menu [Advanced].


LANGKAH 9

Sekarang pilih bahagian [PCI Configuration] lalu tekan [Enter]. Kemudian agar slot USB Anda berfungsi, tetapkan bahagian [USB Function] menjadi [Enabled]. Jika belum, Anda boleh mengubahnya dengan menekan buton [+]. Jika sudah, kembali ke menu [Advanced] dengan menekan buton [Esc].


LANGKAH 10

Selepas itu, Anda dapat mengubah mana-mana penggunaan listrik dari komputer melalui menu[Power] dengan menekan buton [‡]. Namun untuk memudahkannya, tidak perlu mengubah apa-apa dalam pengaturan ini.


LANGKAH 11

Selanjutnya buka menu [Boot]. Untuk memudahkan penginstallan sistem operasi yang akan dilakukan ini, ubahlah pola boot dari PC. Set CD ROM sebagai boot device pertama, kemudian hard disk sebagai boot device kedua dan floopy boot device ketiga, dengan menggunakan tombol [+] atau [-]. Fungsinya, supaya setiap kali booting , komputer akan mencari sistem operasi di CD-ROM terlebih dulu, kemudian baru mencari hard disk , dan floopy


LANGKAH 12

Masukkan CD instalasi Windows XP ke CD-ROM drive . Setelah itu, pilih menu [Exit] [Exit SavingChanges] untuk keluar dari BIOS dan menyimpan pengaturan yang telah Anda telah annda lakukan. Selepas itu komputer akan restart . Dan sekiranya tadi Anda mengaktifkan user password , maka akan muncul kotak password. Anda perlu mengisi password tersebut agar komputer dapat melakukan proses booting

9. hubungan antara leverage operasi dengan leverage keuangan dalam laporan laba rugi?


Rasio Leverages: Operating Leverage Vs Financial Leverage

Leverages adalah kemampuan perusahaan untuk menggunakan aset dari dana pinjaman untuk menciptakan hasil pengembalian (return) yang baik dan mengurangi biaya. Rasio pada Leverages akan menjadi alat pertimbangan penting bagi investor atau kreditur untuk menilai investasi mereka. Itulah mengapa leverage bagi perusahaan dapat berdampak signifikan.

Ada dua jenis Leverage: Operating Leverage dan Financial Leverage. Untuk meningkatkan Financial Leverage, perusahaan dapat meminjam dana melalui penerbitan sekuritas pendapatan tetap (Utang Jangka Panjang, Obligasi, dan lainnya). Sedangkan Operating Leverage dapat digunakan untuk meningkatkan arus kas dan pengembalian-pengembalian (Returns). Dan hal terdebut dapat dicapai melalui peningkatan biaya operasi tetap (Fixed Cost). Kedua metode tersebut bisa berpotensi memunculkan risiko seperti kebangkrutan. Tetapi jika digunakan dengan tepat akan sangat bermanfaat bagi bisnis.

29. jelaskan perbedaan operating system server dengan operating system client !!!


Operating system server merupakan program aplikasi yang bertugas memberikan layanan (service) kepada komputer-komputer yang terhubung melalui suatu jaringan. 
Contoh: Mini Computer, Main Frame, etc. ---> Hardware             DNS, File Server, Application Server, Database Server, Web             Server, Proxy Server, Mail Server, etc ---> Software

Operating system client adalah sebuah sistem komputer atau sebuah proses yang me-request layanan dari server melalui jaringan menggunakan protocol dan menerima respons dari server. Klien merupakan bagian dari sebuah arsitektur jaringan. 
Contoh: PC Workstation ---> Hardware.             Terminal Emulator, Mail Client, etc. ---> Software
