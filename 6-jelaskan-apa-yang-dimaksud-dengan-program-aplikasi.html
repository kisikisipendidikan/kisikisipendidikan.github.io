Pembahasan:

 

1. Apa yg dimaksud booting?

 

Booting adalah proses menyalakan komputer dan memuat perangkat lunak yang diperlukan agar komputer dapat digunakan.

 

2. Sebutkan dan jelaskan jenis booting?

 

-       cold booting: menyalakan komputer dari keadaan benar-benar mati

-       warm booting: menyalakan komputer dari keadaan tidak aktif seperti saat stand by

-       soft booting: proses booting yang dilakukan oleh sistem perangkat lunak komputer

-       hard booting: proses booting yang dilakukan secara paksa denan perangkat keras

-       reboot: proses mematikan komputer lalu menyalakan kembali hingga bisa digunakan lagi

 

3. Bagaimana langkah menggunakan disk defragmenter?

 

Untuk menggunakan disk defragmenter pada sistem operasi Windows dapat digunakan aplikasi bawaan yang terletak di Accessories pada bagian System Tools. lalu, pilih partisi dari hard disk yang akan di-defrag dan jalankan aplikasi. Komputer biasanya perlu tidak digunakan agar partisi hard disk tersebut tidak terisi file dan proses defrag bisa cepat selesai.

 

4. Apa keuntungan mengelola program aplikasi?

 

Keuntunganya adalah kita bisa menginstall program yang kita perlukan, dan mencopot (meng-uninstall) program yang tidak diperlukan.

 

5. Apa yg dimaksud dengan peripheral computer?

 

Peripheral adalah perangkat tambahan yang dapat disambungkan ke komputer untuk melakukan fungsi terntentu, misalnya printer untuk mencetak dokumen atau mouse untuk menggerakkan pointer pada layat.

 

6. Jelaskan 5 tipe-tipe extension?

 

 

- .exe: ekstensi untuk file executable yang bisa dijalankan untuk membuka program aplikasi

- .bmp: file bitmap, berisi data gambar grafis komputer raster yang terdiri dari piksel.

- .jpg: file berisi gambar grafis komputer namun dalam bentuk yang terkompresi

- .txt: file yang berisi text

- .mp3: berisi suara atau musik

                                                                             

7. Jelaskan fungsi Windows Explorer?

 Windows Explorer berfungsi untuk menjelajahi sistem file dan folder yang ada dalam penyimpanan data (hardisk, flash disk, CD) komputer.

Pengertian Jaringan Komputer

Sistem jaringan komputer adalah gabungan atau kumpulan dari beberapa komputer yang dapat diakses secara bersama – sama (seperti floopy disk, CD-ROM, printer, dan sebagainya), dan dapat berhubungan dengan komputer induk sistem lainnya yang letaknya berjauhan.

Tipe-tipe Jaringan Komputer

Ada beberapa tipe jaringan komputer yang umumnya digunakan. Berikut ini beberapa klasifikasi tipe jaringan komputer yang ada :

1. Berdasarkan letak geografis

Local Area Network (LAN), jaringan ini berada pada satu bangunan atau lokasi yang sama, dengan kecepatan transmisi data yang tinggi (mulai dari 10 Mbps ke atas), dan menggunakan peralatan tambahan seperti repeater, hub, dan sebagainya.

2. Berdasarkan arsitektur jaringan

jaringan peer to peer

jaringan berbasis server (server-based network/server-client network)

jaringan hibrid.

(Bagian ini akan dibahas lebih lanjut pada bahasan selanjutnya)

3. Berdasarkan teknologi transmisi

Jaringan switch, merupakan jaringan yang penyampaian informasi dari pengirim ke penerima melalui mesin-mesin perantara atau saluran telepon,

Jaringan broadcast, merupakan jaringan yang penyampaian informasi dari pengirim ke penerima dilakukan secara broadcast (disiarkan ke segala arah) baik melalui saluran kabel maupun saluran tanpa kabel.

a. Server

Sesuai dengan namanya, ini adalah perangkat keras yang berfungsi untuk melayani jaringan dan klien yang terhubung pada jaringan tersebut. Server dapat berupa sistem komputer yang khusus dibuat untuk keperluan tertentu, seperti untuk penggunaan printer secara bersama (print server), untuk hubungan eksternal LAN ke jaringan lain (communication server), dan file server yakni disk yang digunakan secara bersama oleh beberapa klien. Server ini tidak dapat digunakan sebagai klien, karena baik secara hardware maupun software, ia hanya berfungsi untuk mengelola jaringan.
